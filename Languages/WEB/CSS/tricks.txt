### Quick snippets

- QUICK CENTER:
	```css
	html, body{
		height:100%;
	}
	body{
		display:grid;
		place-items:center;
	}
	```

	- ALTERNATIVELY
		padding: auto; /*might help sometimes*/
		
- SCROLL
	- SMOOTH SCROLLING
		```css
		scroll-behavior: smooth /*(as in when you click an anchor tag with href=#myId within the
			page, it'll scroll smoothly) */
		```
		

- NOT DRAGGABLE
	```{css}
	* {
		user-drag: none;
		-webkit-user-drag: none;
		user-select: none;
		-moz-user-select: none;
		-webkit-user-select: none;
		-ms-user-select: none;
	}
	```
	
- SCROLLBARS
	- [more about this](https://www.geeksforgeeks.org/how-to-change-style-of-scrollbar-using-tailwind-css/)
	- EXAMPLE SCROLLBAR:
		```{css}
		::-webkit-scrollbar {
			width: 10px;
		}
		 
		::-webkit-scrollbar-track {
			background: #f1f1f1;
		}
		 
		::-webkit-scrollbar-thumb {
			background: #888;
			border-radius: 5px;
		}
		 
		::-webkit-scrollbar-thumb:hover {
			background: #555;
		}
		```
