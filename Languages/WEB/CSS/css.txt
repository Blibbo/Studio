UNITS OF MEASURE
	rem			<-relative to the font size of the root element of the document (1 is 16px usually)
	em			<-relative to the font size of the current element
	px			<-fucking guess
	%			<-percentage of the parent's size.
					N.B: the parent's size can be a percentage too, but at least one ancestor must have a set size in pixels
