### About JPEGView

- lean photo viewer and editor
- supports pretty much every popular photo format
- released under the [[GPL]]

---

### Configuration

- **Config file:**
	- open the program
	- right click inside the window
	- `Settings/Admin`
	- `Edit user settings...`
- **KeepParameters**
	- keep zoom and filters through image navigation
	- set to `true` or `false`
