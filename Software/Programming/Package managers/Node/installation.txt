NPM SCRIPTS
	make a file (if there isn't one) called package.json and put this inside
	{
		"scripts":{
			"my-script": "my command"
		}
	}
	then to execute it in the console (you must be in the parent folder of package.json)
	use this command
		npm run my-script
	OR the shorthand://NVMNVMNVMNVM it has to exist already, you can't run your own script
		npx my-script
