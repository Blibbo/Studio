**Unity** is a [[game engine]].

## Tech employed

Unity employs:
- [[C-Sharp|C#]] as its primary [[scripting language]].
- [[Mono]] as its C# implementation
- [[Vulkan]] as its graphics [[API]].

## History

Unity was made and is owned by [[Unity Technologies]].
[[Microsoft]] likes this game engine.
